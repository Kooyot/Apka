OPCJA A — GITHUB PAGES (najlepsza):
1. Wrzuć WSZYSTKIE 5 plików do repo na GitHub
2. Settings → Pages → Deploy from branch → main
3. Otwórz https://username.github.io/repo/DIET-MASTER-V7.html
4. Chrome pokaże "Zainstaluj" — kliknij

OPCJA B — TELEFON (szybka):
1. Rozpakuj ZIP na telefonie
2. Przenieś wszystkie pliki do JEDNEGO folderu
3. Użyj "Simple HTTP Server" (Android) lub "Pythonista" (iOS)
4. Otwórz w Chrome → 3 kropki → Zainstaluj

OPCJA C — KOMPUTER (test):
1. Rozpakuj pliki
2. Terminal: python3 -m http.server 8080
3. Otwórz http://localhost:8080/DIET-MASTER-V7.html
4. Chrome pokaże ikonkę instalacji w pasku adresu

WAŻNE:
- Wszystkie 5 plików MUSZĄ być w tym samym folderze
- Service Worker wymaga HTTPS (GitHub Pages) lub localhost
- Manifest musi być dostępny pod ścieżką manifest.json
